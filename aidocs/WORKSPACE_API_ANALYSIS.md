### 2. **API Controller** - [WorkspacesController.cs](Backend/Vault/API/Controllers/WorkspacesController.cs)

```csharp
[ApiController]
[Route("api/[controller]")]
public class WorkspacesController : BaseApiController
{
    private readonly IWorkspaceService _service;

    public WorkspacesController(IWorkspaceService service)
    {
        _service = service;
    }

    [HttpPost]
    public async Task<ActionResult> Post([FromBody] Workspace workspace)
    {
        await _service.CreateWorkspaceAsync(workspace);
        return CreatedAtAction(nameof(Get), new { id = workspace.Id }, workspace);
    }
}
```

**Endpoint:** `POST /api/workspaces`

### 3. **Service Layer** - [WorkspaceService.cs](Backend/Vault/Service/Services/WorkspaceService.cs)

```csharp
public class WorkspaceService : IWorkspaceService
{
    private readonly IUnitOfWork _unitOfWork;

    public WorkspaceService(IUnitOfWork unitOfWork)
    {
        _unitOfWork = unitOfWork;
    }

    public async Task CreateWorkspaceAsync(Workspace workspace)
    {
        await _unitOfWork.Workspaces.AddAsync(workspace);
    }
}
```

## Request/Response Flow

```
Frontend (POST Request)
    ↓
Frontend JSON serialized with camelCase properties
    ↓
WorkspacesController.Post([FromBody] Workspace workspace)
    ↓
WorkspaceService.CreateWorkspaceAsync(workspace)
    ↓
UnitOfWork.Workspaces.AddAsync(workspace)
    ↓
WorkspaceRepository (MongoDB insertion)
    ↓
MongoDB (snake_case fields stored)
    ↓
Response: CreatedAtAction (201 Created)
    ↓
Frontend receives workspace object
```

## Important Notes

1. **Property Naming Convention:**
   - Frontend uses **camelCase** (e.g., `tenantId`, `retentionPolicyId`)
   - Backend uses **PascalCase** (e.g., `TenantId`, `RetentionPolicyId`)
   - MongoDB stores as **snake_case** (e.g., `tenant_id`, `retention_policy_id`)

2. **ObjectId References:**
   - Properties like `Id`, `TenantId`, `RetentionPolicyId`, `CreatedBy`, `UpdatedBy` are MongoDB ObjectIds
   - Use `[BsonRepresentation(BsonType.ObjectId)]` attribute in C#

3. **Auto-generated Fields:**
   - `Id` is auto-generated if not provided
   - `CreatedAt` should be set at creation
   - `UpdatedAt` should be updated on modifications

4. **Validation:**
   - No validation is currently shown in the code
   - Add validation in service layer or controller if needed

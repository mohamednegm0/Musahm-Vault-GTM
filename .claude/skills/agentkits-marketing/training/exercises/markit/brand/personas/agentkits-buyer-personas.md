## Persona 1: Solo Sam

### Demographics

| Attribute | Detail |
|-----------|--------|
| **Name** | Solo Sam |
| **Age** | 25-35 |
| **Role** | Solopreneur / Indie Hacker |
| **Company Size** | 1 person (bootstrapped) |
| **Location** | Remote, global |
| **Income** | $50K-150K ARR |

### Psychographics

**Goals:**
- Ship products fast, validate ideas
- Grow without hiring a marketing team
- Automate repetitive marketing tasks
- Build in public, gain followers

**Frustrations:**
- "Marketing takes time away from building"
- "Can't afford marketing tools or freelancers"
- "Don't know what to write or post"
- "Inconsistent content, no schedule"

**Motivations:**
- Independence and flexibility
- Proving it can be done solo
- Building sustainable income
- Learning new skills

### Behavior Patterns

**Tools Used:**
- VS Code / Cursor
- Twitter/X for building in public
- Product Hunt for launches
- Stripe for payments

**Content Consumption:**
- Indie Hackers forum
- Twitter tech threads
- Podcasts (Indie Hackers, MFM)
- YouTube tutorials

**Decision Factors:**
1. Free / Open source (critical)
2. Fast to learn
3. Works in terminal/CLI
4. Community support

### Messaging

**Value Proposition:**
"Full marketing team in your terminal. Free forever."

**Key Messages:**
- "Run one command, get marketing content"
- "No subscriptions, no API keys"
- "Built for builders like you"

**Channels:**
- Twitter/X
- Product Hunt
- Indie Hackers
- Dev.to

## How to Use These Personas

### Content Creation

When writing content, ask:
- "Would Solo Sam understand this?"
- "Does this save Marketer Maya time?"
- "Is this technical enough for Founder Felix?"

### Feature Prioritization

- Solo Sam: Free tier, quick wins
- Marketer Maya: Team features, integrations
- Founder Felix: CLI experience, documentation

### Channel Strategy

| Persona | Primary | Secondary |
|---------|---------|-----------|
| Solo Sam | Twitter, Product Hunt | Reddit, Discord |
| Marketer Maya | LinkedIn, Email | Webinars, Blogs |
| Founder Felix | Hacker News, GitHub | Twitter, Podcasts |

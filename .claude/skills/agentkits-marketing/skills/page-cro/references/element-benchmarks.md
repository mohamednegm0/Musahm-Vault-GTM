# Page Element Benchmarks & Optimization Data

Research-backed benchmarks for page elements. Sources: [WebFX](https://www.webfx.com/blog/conversion-rate-optimization/cro-trends/), [Fibr](https://fibr.ai/conversion-rate-optimization/best-practices), [CXL](https://cxl.com/).

## Page Speed Impact

| Load Time | Bounce Rate | Conversion Drop |
|-----------|-------------|-----------------|
| 1-2 seconds | Baseline | Baseline |
| 2-3 seconds | +10% | -7% |
| 3-4 seconds | +25% | -13% |
| 4-5 seconds | +38% | -20% |
| 5-6 seconds | +50% | -25% |
| 6+ seconds | +90% | -40%+ |

**Key Insight:** Each 1-second delay = 7% conversion loss

### Core Web Vitals Targets
| Metric | Good | Needs Improvement | Poor |
|--------|------|-------------------|------|
| LCP (Largest Contentful Paint) | <2.5s | 2.5-4s | >4s |
| FID (First Input Delay) | <100ms | 100-300ms | >300ms |
| CLS (Cumulative Layout Shift) | <0.1 | 0.1-0.25 | >0.25 |

## Headline Optimization

### Length Impact
| Headline Length | Performance |
|-----------------|-------------|
| 6 words | Optimal recall |
| 8-12 words | Good balance |
| 13+ words | Decreasing engagement |

### Headline Formulas (by Performance)
| Formula | Avg Lift | Example |
|---------|----------|---------|
| Number + Adjective + Noun | +36% | "7 Proven Ways to..." |
| How to + Benefit | +28% | "How to Double Revenue" |
| Question + Pain Point | +24% | "Tired of Manual Tasks?" |
| Command + Benefit | +21% | "Get More Leads Today" |
| Negative + Solution | +18% | "Stop Losing Customers" |

### Power Words in Headlines
**Urgency:** Now, Today, Fast, Quick, Instant
**Exclusivity:** Secret, Exclusive, Limited, VIP
**Value:** Free, Save, Bonus, Extra, Guaranteed
**Trust:** Proven, Research-backed, Expert

## CTA Optimization

### Button Color Impact (Context-Dependent)
| Color | Best For | Caution |
|-------|----------|---------|
| Orange | Urgency, action | Can feel cheap |
| Green | Positive action, go | Overused |
| Blue | Trust, professional | Low urgency |
| Red | Urgency, sales | Aggressive |

**Key Insight:** Contrast matters more than color. CTA should be most prominent element.

### CTA Text Performance
| CTA Text | Performance vs "Submit" |
|----------|------------------------|
| "Get Started" | +38% |
| "Try For Free" | +23% |
| "Start My Trial" | +90% |
| "Claim My Spot" | +42% |
| "Get [Benefit]" | +33% |

### First Person vs Second Person
| Type | Example | Performance |
|------|---------|-------------|
| First Person | "Start My Trial" | +90% |
| Second Person | "Start Your Trial" | Baseline |

## Pricing Page Specifics

### Number of Tiers
| Tiers | Best For | Notes |
|-------|----------|-------|
| 2 | Simple products | Binary choice |
| 3 | Most SaaS | Decoy effect works |
| 4 | Complex products | Enterprise segment |
| 5+ | Avoid | Decision paralysis |

### Decoy Effect (Price Anchoring)
Middle tier selected 40% more often when:
- Lowest tier lacks key feature
- Highest tier is 2-3x middle price
- Middle labeled "Popular" or "Recommended"

### Annual vs Monthly Toggle
| Default | Conversion Impact |
|---------|------------------|
| Monthly first | More signups, lower LTV |
| Annual first | Fewer signups, higher LTV |
| Annual with savings % | +25% annual selection |

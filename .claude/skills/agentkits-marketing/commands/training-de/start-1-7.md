### Lektionsübersicht

---

**Modul 1.7: Navigation & Suche**

Effiziente Navigation ist entscheidend, wenn deine Marketing-Projekte wachsen. Lass uns das Finden von Befehlen und die Verwaltung von Workflows meistern.

**Dauer:** ~20 Minuten

---

### Schritt 1: Befehle finden

Übe das Entdecken von Befehlen:

```
What commands are available for campaigns?
```

```
Show me all SEO-related commands
```

```
What commands help with email marketing?
```

### Schritt 2: Befehlskategorien-Referenz

Überprüfe alle Befehlskategorien:

| Kategorie | Befehle | Anwendungsfälle |
|----------|----------|-----------|
| `/campaign:*` | plan, brief, analyze, calendar | Kampagnenmanagement |
| `/content:*` | blog, social, email, landing, ads | Content-Erstellung |
| `/seo:*` | keywords, audit, optimize, competitor | SEO-Optimierung |
| `/analytics:*` | roi, funnel, report | Performance-Tracking |
| `/sales:*` | pitch, outreach, battlecard, qualify | Sales Enablement |
| `/leads:*` | score, nurture, qualify | Lead-Management |
| `/crm:*` | sequence, segment, score, lifecycle | CRM-Automatisierung |
| `/sequence:*` | welcome, nurture, re-engage | E-Mail-Sequenzen |
| `/social:*` | engage, viral, schedule | Social Media |
| `/research:*` | market, persona, trend | Marktforschung |
| `/brand:*` | voice, book, assets | Markenmanagement |
| `/ops:*` | daily, weekly, monthly | Betrieb |
| `/report:*` | weekly, monthly | Berichterstattung |
| `/checklist:*` | campaign-launch, seo-weekly | Checklisten |

### Schritt 3: Agenten finden

Übe das Entdecken von Agenten:

```
Which agent handles lead scoring?
```

```
What agent should I use for competitive analysis?
```

```
Show me the reviewer agents
```

### Schritt 4: Dokumentation finden

Übe das Finden relevanter Dokumentation:

```
What documentation covers brand voice?
```

```
Where can I find campaign playbooks?
```

```
Show me the content style guide
```

### Schritt 5: Schnellreferenz-Befehle

Teile wichtige Befehlsabkürzungen:

**Schneller Content:**
```
/content:fast "quick headline for social post"
```

**Guter Content:**
```
/content:good "detailed blog post with research"
```

**Brainstorming:**
```
/brainstorm "marketing channel ideas for B2B SaaS"
```

### Schritt 6: Betriebsbefehle

Übe operative Workflows:

**Tägliche Operationen:**
```
/ops:daily "content creation"
```

**Wöchentliche Überprüfung:**
```
/ops:weekly "current week dates"
```

**Kampagnen-Checklisten:**
```
/checklist:campaign-launch "Q1 Product Launch" "next week"
```

### Schritt 7: Vollständiger Befehls-Workflow

Demonstriere einen vollständigen Workflow:

```
1. /research:market "B2B productivity software"
2. /campaign:plan "Q2 Growth Campaign"
3. /seo:keywords "team productivity"
4. /content:blog "productivity topic" "target keyword"
5. /seo:optimize "the blog post" "keyword"
6. /content:social "blog topic" "linkedin"
7. /sequence:nurture "product" "blog readers"
8. /analytics:report "Q2" "all channels"
```

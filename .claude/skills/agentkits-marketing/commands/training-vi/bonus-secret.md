## Tư Duy 10x Marketer

Hầu hết marketers làm việc **linearly** - mỗi campaign đòi hỏi effort như nhau.

**10x Marketers** làm việc **exponentially** - mỗi campaign làm campaign tiếp theo nhanh hơn và tốt hơn.

Bí mật: **Systematic compounding**

## Bốn Trụ Cột

### Trụ Cột 1: Command Mastery

Biết commands nào chain cùng nhau:

**Campaign Launch Workflow:**
```
1. /research:market → Hiểu thị trường
2. /research:persona → Định nghĩa audience
3. /campaign:plan → Strategic plan
4. /campaign:brief → Creative direction
5. /campaign:calendar → Content schedule
6. /seo:keywords → SEO foundation
7. /content:blog → Pillar content
8. /content:social → Distribution content
9. /sequence:welcome → Lead nurture
10. /analytics:funnel → Measurement
```

**Content Production Workflow:**
```
1. /seo:keywords → Tìm opportunities
2. /content:blog → Tạo content
3. /seo:optimize → Tối ưu cho search
4. /content:social → Repurpose cho social
5. Review với agents → Quality check
```

### Trụ Cột 2: Agent Orchestration

Sử dụng agents một cách chiến lược:

**Cho Research Phase:**
- `researcher` → Market understanding
- `attraction-specialist` → Competitive intel

**Cho Creation Phase:**
- `brainstormer` → Campaign concepts
- `copywriter` → Content creation
- `planner` → Strategic structure

**Cho Review Phase:**
- `brand-voice-guardian` → Voice check
- `conversion-optimizer` → CRO review
- `seo-specialist` → SEO validation
- Persona reviewers → Audience fit

**Cho Optimization Phase:**
- `lead-qualifier` → Lead quality
- `email-wizard` → Sequence optimization
- `continuity-specialist` → Retention

### Trụ Cột 3: Documentation Leverage

Sử dụng documentation để compound knowledge:

**Trước mỗi campaign:**
```
1. Đọc docs/campaign-playbooks.md
2. Đọc docs/content-style-guide.md
3. Đọc docs/channel-strategies.md
4. Review pattern library của bạn
```

**Sau mỗi campaign:**
```
1. Update pattern library với learnings
2. Document winning approaches
3. Ghi chú điều gì nên tránh
4. Track performance trends
```

## Hệ Thống Thực Thi 10x

### Hàng Ngày: Quick Wins (15 phút)
```
/ops:daily "focus area"
```
- Kiểm tra metrics
- Tạo 1 piece of content
- Update pattern library

### Hàng Tuần: Strategic Review (1 giờ)
```
/ops:weekly "tuần hiện tại"
```
- Review campaign performance
- Lập kế hoạch content tuần tới
- Update sequences dựa trên data

### Hàng Tháng: Deep Analysis (2 giờ)
```
/ops:monthly "tháng hiện tại"
/analytics:report "all campaigns" "full analysis"
```
- Comprehensive performance review
- Update documentation
- Refine strategies

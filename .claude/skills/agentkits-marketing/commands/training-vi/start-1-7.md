| Category | Commands | Use Cases |

| `/seo:*` | keywords, audit, optimize, competitor | Tối ưu SEO |
| `/analytics:*` | roi, funnel, report | Performance tracking |
| `/sales:*` | pitch, outreach, battlecard, qualify | Hỗ trợ bán hàng |
| `/leads:*` | score, nurture, qualify | Quản lý lead |

| `/sequence:*` | welcome, nurture, re-engage | Email sequences |

### Bước 5: Quick Reference Commands

Chia sẻ shortcuts command thiết yếu:

**Fast Content:**
```
/content:fast "quick headline cho social post"
```

**Good Content:**
```
/content:good "detailed blog post với research"
```

**Brainstorming:**
```
/brainstorm "ý tưởng marketing channel cho B2B SaaS"
```

### Bước 6: Operational Commands

Thực hành operational workflows:

**Daily Operations:**
```
/ops:daily "content creation"
```

**Weekly Review:**
```
/ops:weekly "ngày tuần hiện tại"
```

**Campaign Checklists:**
```
/checklist:campaign-launch "Q1 Product Launch" "tuần tới"
```

### Bước 7: Complete Command Workflow

Trình bày workflow hoàn chỉnh:

```
1. /research:market "B2B productivity software"
2. /campaign:plan "Q2 Growth Campaign"
3. /seo:keywords "team productivity"
4. /content:blog "productivity topic" "target keyword"
5. /seo:optimize "blog post" "keyword"
6. /content:social "blog topic" "linkedin"
7. /sequence:nurture "product" "blog readers"
8. /analytics:report "Q2" "all channels"
```

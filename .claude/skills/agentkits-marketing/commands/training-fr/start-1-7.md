**Module 1.7 : Navigation & Recherche**

Une navigation efficace est cruciale à mesure que vos projets marketing se développent. Maîtrisons la recherche de commandes et la gestion des flux de travail.

**Durée :** ~20 minutes

### Étape 1 : Trouver les Commandes

Pratiquez la découverte de commandes :

```
What commands are available for campaigns?
```

```
Show me all SEO-related commands
```

```
What commands help with email marketing?
```

### Étape 2 : Référence des Catégories de Commandes

Passez en revue toutes les catégories de commandes :

| Catégorie | Commandes | Cas d'Usage |
|----------|----------|-----------|
| `/campaign:*` | plan, brief, analyze, calendar | Gestion de campagne |
| `/content:*` | blog, social, email, landing, ads | Création de contenu |
| `/seo:*` | keywords, audit, optimize, competitor | Optimisation SEO |
| `/analytics:*` | roi, funnel, report | Suivi des performances |
| `/sales:*` | pitch, outreach, battlecard, qualify | Activation des ventes |
| `/leads:*` | score, nurture, qualify | Gestion des prospects |
| `/crm:*` | sequence, segment, score, lifecycle | Automatisation CRM |
| `/sequence:*` | welcome, nurture, re-engage | Séquences d'emails |
| `/social:*` | engage, viral, schedule | Médias sociaux |
| `/research:*` | market, persona, trend | Étude de marché |
| `/brand:*` | voice, book, assets | Gestion de marque |
| `/ops:*` | daily, weekly, monthly | Opérations |
| `/report:*` | weekly, monthly | Reporting |
| `/checklist:*` | campaign-launch, seo-weekly | Listes de contrôle |

### Étape 4 : Trouver la Documentation

Pratiquez la recherche de documentation pertinente :

```

```

```

```

```
Show me the content style guide
```

### Étape 5 : Commandes de Référence Rapide

Partagez les raccourcis de commandes essentiels :

**Contenu Rapide :**
```
/content:fast "quick headline for social post"
```

**Bon Contenu :**
```
/content:good "detailed blog post with research"
```

**Brainstorming :**
```
/brainstorm "marketing channel ideas for B2B SaaS"
```

### Étape 6 : Commandes Opérationnelles

Pratiquez les flux de travail opérationnels :

**Opérations Quotidiennes :**
```
/ops:daily "content creation"
```

**Revue Hebdomadaire :**
```
/ops:weekly "current week dates"
```

**Listes de Contrôle de Campagne :**
```
/checklist:campaign-launch "Q1 Product Launch" "next week"
```

### Étape 7 : Flux de Travail Complet de Commandes

Démontrez un flux de travail complet :

```
1. /research:market "B2B productivity software"
2. /campaign:plan "Q2 Growth Campaign"
3. /seo:keywords "team productivity"
4. /content:blog "productivity topic" "target keyword"
5. /seo:optimize "the blog post" "keyword"
6. /content:social "blog topic" "linkedin"
7. /sequence:nurture "product" "blog readers"
8. /analytics:report "Q2" "all channels"
```

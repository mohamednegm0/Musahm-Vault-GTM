### Descripción de la Lección

---

**Módulo 2.6: Optimización SEO**

El SEO impulsa un crecimiento sostenible a largo plazo. Dominemos la investigación de palabras clave, la optimización de contenido y el SEO técnico.

**Duración:** ~40 minutos

---

### Paso 1: Investigación de Palabras Clave

Usa el comando de investigación de palabras clave:

```
/seo:keywords "remote team productivity"
```

```
/seo:keywords "team focus time software"
```

```
/seo:keywords "coordination tools for remote teams"
```

Revisa las oportunidades de palabras clave:
- Estimaciones de volumen de búsqueda
- Niveles de competencia
- Clasificación de intención
- Grupos de palabras clave

### Paso 2: Análisis SEO de Competidores

Analiza el SEO de competidores:

```
/seo:competitor "rescuetime.com"
```

```
/seo:competitor "freedom.to"
```

Identifica:
- Palabras clave por las que rankean
- Brechas de contenido
- Oportunidades de backlinks
- Fortalezas/debilidades técnicas

### Paso 3: Auditoría SEO

Ejecuta una auditoría integral:

```
/seo:audit "agentkits.com"
```

Revisa las áreas de auditoría:
- SEO técnico (rastreabilidad, velocidad, móvil)
- SEO on-page (títulos, meta, encabezados)
- Calidad del contenido (profundidad, relevancia)
- Perfil de backlinks

### Paso 4: Optimización de Contenido

Optimiza el contenido existente:

```
/seo:optimize "blog post about team productivity" "remote team productivity"
```

La optimización incluye:
- Mejora de la etiqueta de título
- Meta descripción
- Estructura de encabezados (H1, H2, H3)
- Colocación de palabras clave
- Sugerencias de enlaces internos
- Texto alt de imágenes

### Paso 5: Crear Contenido SEO

Usa el comando de blog con enfoque SEO:

```
/content:blog "The Complete Guide to Remote Team Productivity" "remote team productivity"
```

Luego optimiza:

```
/seo:optimize "the blog post" "remote team productivity"
```

### Paso 6: Estrategia de Fragmentos Destacados

Optimiza para fragmentos destacados:

```
Create content optimized for featured snippets:

Target query: "How to improve remote team productivity"

PARAGRAPH SNIPPET (40-60 words):
Direct answer format

LIST SNIPPET (numbered steps):
Step-by-step format

TABLE SNIPPET:
Comparison format
```

### Paso 7: Lista de Verificación SEO Técnico

Usa la lista de verificación SEO:

```
/checklist:seo-weekly "agentkits.com"
```

Tareas SEO semanales:
- Verificación de errores de rastreo
- Monitoreo de velocidad de página
- Usabilidad móvil
- Estado del índice
- Seguimiento de palabras clave

### Paso 8: Estrategia Pilar-Cluster

Crea un plan de contenido pilar:

```
Create pillar-cluster SEO strategy:

PILLAR PAGE: "The Complete Guide to Remote Team Productivity"
- 3000-5000 words, comprehensive

CLUSTER CONTENT (10 posts):
1. "How to Schedule Team Focus Time" → team focus time
2. "Reducing Meetings Without Losing Alignment" → reduce meetings remote
3. "Async Communication Best Practices" → async communication
4. "Deep Work for Remote Teams" → deep work teams
5. "Team Productivity Metrics That Matter" → productivity metrics
...

INTERNAL LINKING:
- Each cluster links to pillar
- Pillar links to all clusters
- Cross-links between related clusters
```

### Paso 9: Calendario de Contenidos con SEO

Crea un calendario enfocado en SEO:

```
/campaign:calendar "12 weeks - SEO content strategy - target keywords: remote team productivity, team focus time, coordination software"
```

Mapea el contenido a palabras clave:
- Semana 1: [Palabra clave] - [Título] - [Tipo]
- Semana 2: [Palabra clave] - [Título] - [Tipo]
...

### Paso 10: Medición SEO

Rastrea el rendimiento SEO:

```
/analytics:report "SEO performance" "organic traffic, keyword rankings, conversions"
```

Métricas clave:
- Tendencia de tráfico orgánico
- Cambios de posición de palabras clave
- Tasa de conversión orgánica
- Rendimiento de páginas

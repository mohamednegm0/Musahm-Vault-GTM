1. Start with `positioning-statement.md` -- source of truth for all messaging
2. Check `voice-guide.md` before writing any customer-facing content
3. Run `brand-governance-checklist.md` 14-item gate before publishing ANYTHING
4. Use `battle-cards.md` for competitive situations
5. Use `meddic-checklist.md` to qualify every deal
6. Use `cxo-value-story.md` for executive meetings
7. Use `demo-script.md` for product demonstrations
8. Use `account-tiering-matrix.md` to prioritize pipeline
9. Use `pricing-tiers.md` for quoting (fill SAR amounts first)
10. Use `crisis-playbook.md` when things go wrong
11. Use `win-loss-template.md` after every deal outcome

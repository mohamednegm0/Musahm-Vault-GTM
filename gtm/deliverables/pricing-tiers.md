## Segmentation: Who Buys What

| Segment          | Company Profile                                                                      | Primary Need                                       | Expected Tier |
| ---------------- | ------------------------------------------------------------------------------------ | -------------------------------------------------- | ------------- |
| **Starter**      | LLC, 5-20 employees, 1-3 shareholders, no board committees                           | Basic governance + document storage                | Starter       |
| **Professional** | Closed JSC, 20-100 employees, active board, 5-20 shareholders                        | Full governance + Vault + shareholder registry     | Professional  |
| **Enterprise**   | Listed or large closed JSC, 100-500 employees, multiple committees, 20+ shareholders | Everything + advanced workflows + priority support | Enterprise    |

---

## Pricing Pillars (What Determines Price)

| Pillar       | What It Measures                     | Why It Matters                                                         |
| ------------ | ------------------------------------ | ---------------------------------------------------------------------- |
| **Users**    | Number of active users (named seats) | Direct cost driver: each user needs auth, storage, audit trail entries |
| **Storage**  | Total document storage (GB)          | Vault cost scales with document volume                                 |
| **Features** | Governance depth (modules unlocked)  | Higher tiers unlock shareholder registry, workflows, AI extraction     |
| **Support**  | Response SLA and channel             | Enterprise needs guaranteed response times                             |

---

## Pricing Metric: Per-User/Month, Billed Annually

The pricing metric is **per named user per month**, billed annually. This aligns cost with value: more people governing = more value = higher willingness to pay. Monthly billing available at a premium.

## Sales Guidance

### When to Recommend Each Tier

| Signal                                                  | Recommended Tier           | Reasoning                                                        |
| ------------------------------------------------------- | -------------------------- | ---------------------------------------------------------------- |
| "We just need to store board documents"                 | Starter                    | Basic need, small team, expand later                             |
| "We need shareholder registry and committee management" | Professional               | Core governance needs                                            |
| "We have 20+ shareholders and multiple committees"      | Enterprise                 | Scale and complexity require full feature set                    |
| "We currently use Majles/Ebana/Diligent"                | Professional or Enterprise | Positioned as replacement with more value at lower cost          |
| "We just want to try it"                                | Professional (beta)        | Beta is free -- get them on Professional features to build habit |

### Objection Handling: Price

| Objection                            | Response                                                                                                                                                  |
| ------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------- |
| "Too expensive"                      | "What does it cost when the auditor asks for a board resolution and you spend 3 days searching? One governance incident costs more than years of Musahm." |
| "We can use SharePoint/Google Drive" | "Those store files. Vault governs documents. Per-document access control, audit trail, approval workflows -- none of that exists in file storage tools."  |
| "Can we get a discount?"             | "Annual billing saves [X]%. For 10+ seats, ask about volume pricing."                                                                                     |
| "We need to see value first"         | "That is why we offer 30 beta seats. Full Professional features, no commitment. Try it with real documents."                                              |

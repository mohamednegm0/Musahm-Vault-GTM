## Design Direction Notes (for design team)

| Aspect                | Guidance                                                                                                                                                                                                                                                                                                       |
| --------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Brand concept**     | Concept 3: KSA-Native-Tech. Confident, institutional, proudly Saudi. Not startup energy.                                                                                                                                                                                                                       |
| **Primary visual**    | Abstract representation of document protection: vault door with organized files visible inside, or shield icon enclosing document layers. Avoid generic "cloud" or "lock" illustrations. The visual should feel institutional and Saudi, not Silicon Valley.                                                   |
| **Color palette**     | **Primary:** #1B4332 (dark green). **Accent:** #c3924d (gold). **Secondary:** #2563EB (oasis blue). **Background:** #f9fafb. **Text:** #111827. Ensure strong contrast for text readability on all formats. Gold used for accents, CTA buttons, and brand line. Green dominates the background or header area. |
| **Typography**        | Arabic: a clean Naskh or geometric sans (e.g., IBM Plex Arabic, Noto Sans Arabic). English: geometric sans that pairs well. Headline weight: Bold/Black. Body: Regular.                                                                                                                                        |
| **Layout**            | Headline top-center (large), subheadline below (medium), body text center (small), CTA button bottom-center (prominent, gold #c3924d background with #111827 text), brand line bottom-right (small).                                                                                                           |
| **RTL consideration** | Arabic version must be fully RTL: right-aligned text, reversed layout elements.                                                                                                                                                                                                                                |
| **Logo placement**    | Musahm logo top-right (Arabic) or top-left (English). Vault sub-brand lockup below main logo.                                                                                                                                                                                                                  |
| **Formats needed**    | 1080x1080 (social square), 1200x628 (LinkedIn/Twitter landscape), 1080x1920 (story/vertical), print A4                                                                                                                                                                                                         |
| **No-go**             | No stock photos of generic offices. No handshake imagery. No globe icons. No "MENA" or "Middle East" references. No startup-energy design language. Keep it clean and product-focused.                                                                                                                         |
